How to Use the Network Explorer

Filter the network

Use category filters, minimum frequency, and minimum edge weight to control which hashtags and relationships are visible.

Inspect a hashtag

Hover over a node to see its category, frequency, and strongest connections.

Pin nodes

Drag a node to fix it in place. Double-click a node to unpin it. Use Clear all pinned nodes to reset the layout.

Compare tags

Add two or more hashtags in Compare Tags, then use Show comparison layout to see which tags connect to each selected term.

Hide dominant tags

Use Hide Tags to remove very central hashtags, such as 능력녀 or 로맨스판타지, and reveal smaller patterns.

Read the edges

An edge means that two hashtags appear in the same work. A thicker edge means that the co-occurrence appears in more works.
